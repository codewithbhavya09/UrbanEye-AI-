# Screenshots Folder

This folder is used to store screenshots of the UrbanEye AI application
for display in the main project `README.md` file.

## Suggested screenshots to add

1. `home.png` – The main landing page of the app
2. `register.png` – The user registration page
3. `login.png` – The user login page
4. `report_issue.png` – The issue reporting page showing a detected issue
5. `complaint_history.png` – The complaint history page
6. `dashboard.png` – The dashboard page showing charts
7. `admin_panel.png` – The admin panel showing complaint status updates

## How to add screenshots

1. Run the app using `streamlit run app/Home.py`
2. Take a screenshot of each page
3. Save the screenshots in this folder using the filenames listed above
4. They will automatically appear in the main `README.md` file, since it
   already references these file paths using Markdown image syntax
